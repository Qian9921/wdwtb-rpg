## 1. 评委 3 分钟可玩路径（必过）

- [ ] `npm install && npm run dev` 打开本地页
- [ ] 开始 → 捏人 → 测评（可随意）→ 职业大厅
- [ ] 任选 **程序员（★完整版）· 开发**（默认推荐路径）

**备选职业冒烟（各 ≤1 分钟）**：产品 / 行政 / 任一迷你完整（设计师或律师）能进办公室、见名册 NPC、工位可交互。

## 5. 已知限制（诚实，非阻塞演示）

1. **WorldScene 仍偏大**（~2700 行）：已抽出 `StoryProgress` / `MinigameFlavor`，地图与输入仍集中在场景文件；并行改请先对齐，避免互踩。  
2. **主线 e2e 以程序员为主**：其它职业靠 validate + 任务链单测 + 手工冒烟。  
3. **AI 结局/NPC 记忆**：无 Key 时降级，不阻塞通关。  
4. **Debug 代码题** 仍主要服务程序员；非程序员用文本挑错池（`debug_puzzles.pools.*`）。  
5. **「第一名」** 由黑客松评委决定，不在自动化范围内。
