# 创作者初衷 · 产品三柱（协作宪法）

> 本文档是《你想成为谁》的**北极星**。  
> 一切功能、美术、数值、AI、比赛演示，都服务于此；不得用工期或炫技覆盖初衷。  
> 协作方（含多 Agent）改代码前应先读本节。

---

## 1. 为谁做

**刚毕业、或即将毕业、对职业仍感到迷茫的大学生。**

他们需要的不是又一份「热门职业排行」，也不是空洞鸡汤，而是：

- 能**安全试用**不同职业生活与环境的机会；
- 在试过之后，对「适不适合我、喜不喜欢」有更清楚的体感与语言。

---

## 2. 为什么做（初衷原话级表述）

**让刚毕业的大学生可以真实地体验不同的职业生活和环境，帮助他们走出迷茫，找到真正适合自己、自己也喜欢的职业。**

补充气质（与初衷一体，不可拆成「只做治愈文案」）：

> 你不是要成为一个正确的人，而是要认出那个本来的你。

- **适合自己**：兴趣、气质、承受方式与职业环境是否咬合；  
- **喜欢**：过几天那种日子时，身体与心情是否愿意再来；  
- **认出**：试完之后能说清「我是谁、我怕什么、什么在推我」。

本作为：

> **用像素 RPG 做载体的、面向毕业生的体验式职业探索与自我澄清工具。**

---

## 3. 产品三柱（不可拆）

三柱必须同时存在。砍掉任一柱，作品就滑向「测验网页」「上班模拟」或「纯剧情」之一，偏离初衷。

```
        ┌───────────── 测评 Assessment ─────────────┐
        │  科学坐标系：你大概是谁、兴趣落在哪一带     │
        │  RIASEC / 大五人格→MBTI 倾向 / 情境题       │
        │  初见画像 · 职业细分推荐（可改）             │
        └──────────────────┬────────────────────────┘
                           │ 带着画像出发
        ┌──────────────────▼────────────────────────┐
        │  体验 Experience                           │
        │  过几天那种人的日子：办公室·同事·任务·      │
        │  工位·压力·下班·家人·选择轨迹               │
        └──────────────────┬────────────────────────┘
                           │ 带着体感回来
        ┌──────────────────▼────────────────────────┐
        │  报告 Report                               │
        │  把体验翻译成可带走的洞见：                  │
        │  心之画像 · 驱动力/消耗源/压力关系/         │
        │  隐藏模式 · 职业契合 · 一句专属的话         │
        └───────────────────────────────────────────┘
```

| 柱 | 职责 | 玩家带走什么 | 实现锚点（现状） |
|---|---|---|---|
| **测评** | 严肃对待「适合」：用研究成果给方向，不用术语吓退 | 被量过、被看见的「初见坐标」 | `OpeningScene` + `public/data/assessment.json`（霍兰德 RIASEC + 大五；玩家不见术语）→ profile（holland/mbti）→ Hub 细分推荐 |
| **体验** | 严肃对待「喜欢 / 吃不消」：环境与日常必须像真的过日子 | 「这种班我待不待得住」的体感 | WorldScene 工作日循环、任务链、工单、名册 NPC、通勤/家、状态条、内心独白 |
| **报告** | 严肃对待「不白玩」：体验必须被翻译成自我理解 | 可复述的洞见与下一步思考 | `EndingScene`《心之画像》（AI 生成 + 降级模板；吃状态 + **选择轨迹**） |

### 3.1 设计伦理（与三柱同级）

- **不替人做决定**：推荐可改，契合度诚实但不劝退。  
- **专业要对得起学术，呈现要对得起迷茫的人**：情境题承载模型，报告禁止空泛贴标签。  
- **AI 是助手不是主菜**：无网/失败时仍须有完整模板报告；手写与数据驱动的体验主体不可被模型替代。  
- **疗愈不是猎奇**：可写过劳与迷茫，不为贩卖焦虑。

---

## 4. 决策过滤器（改需求时自问）

1. 这是否帮助大学生**更真实地体验**某种职业生活/环境？  
2. 这是否帮助他们**更清楚适不适合、喜不喜欢**？  
3. 这是否落在 **测评 / 体验 / 报告** 至少一柱，且不拆掉另外两柱？  

若三问都偏「否」，再酷也不做。
